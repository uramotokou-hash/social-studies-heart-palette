# 社会科 ふり返り心パレット

小学校第5学年社会科において、
「分かったこと」「感情」「感情の理由」
「自分とのつながり」を記録するWebアプリです。

## 使い方

- `index.html` をブラウザで開くだけで動きます（インストール・ビルド不要）
- 記録は端末内（localStorage）に保存されます
- 教師用画面：Home画面の「🍎 先生用ログイン」→ 暗証番号 2468

## きろくを先生のスプレッドシートに自動で集める

各タブレットの記録を、先生のGoogleスプレッドシートへ自動送信できます。
設定手順は [docs/先生用セットアップ.md](docs/先生用セットアップ.md) を見てください。
