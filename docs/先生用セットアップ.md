# 先生用セットアップ：きろくをスプレッドシートに自動で集める

児童がタブレットで「保存する」を押すと、先生のGoogleスプレッドシートに1件1行で自動的に届くようにする設定です。

- 使うのは **学校のGoogleアカウントだけ**（無料）
- 児童のきろくは **先生のGoogleドライブの中** にだけ保存されます
- 設定は最初の1回だけ。だいたい **5〜10分** で終わります

---

## 手順1：スプレッドシートを作る

1. Googleドライブを開く → **［新規］→［Googleスプレッドシート］**
2. 名前を「**心パレットきろく**」などに変えます

## 手順2：Apps Script（アップススクリプト）を開く

「Apps Script」は、スプレッドシートに小さなプログラムを付けられるGoogleの無料機能です。

1. スプレッドシートのメニューで **［拡張機能］→［Apps Script］** をクリック
2. 新しいタブでエディタ（プログラムを書く画面）が開きます

## 手順3：コードを貼り付ける

1. 画面に最初から書いてある `function myFunction() {...}` を **全部消します**
2. 下のコードを **全部コピーして貼り付け** ます
3. 上の💾（保存）を押します

```javascript
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var records = data.records || [];
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName('きろく') || ss.insertSheet('きろく');
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(['受信日時','保存日時','学習日','児童番号','単元名',
        '分かったこと','えらんだ気もち','モヤモヤ・スッキリ','気もちの理由','つながり','ID']);
    }
    // すでに受け取ったID（同じきろくを二重に書かないため）
    var ids = {};
    if (sheet.getLastRow() > 1) {
      sheet.getRange(2, 11, sheet.getLastRow() - 1, 1).getValues()
        .forEach(function(row){ ids[row[0]] = true; });
    }
    var added = 0;
    records.forEach(function(r) {
      if (!r || !r.id || ids[r.id]) return;
      sheet.appendRow([new Date(), r.savedAt || '', r.date || '', r.studentNo || '',
        r.unit || '', r.understanding || '', r.emotionText || '', r.scaleText || '',
        r.reason || '', r.connections || '', r.id]);
      ids[r.id] = true;
      added++;
    });
    return ContentService
      .createTextOutput(JSON.stringify({ ok: true, added: added }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ ok: false, error: String(err) }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// 接続テスト用（アプリの「接続テスト」ボタンが使います）
function doGet() {
  return ContentService
    .createTextOutput(JSON.stringify({ ok: true, ping: 'heart-palette' }))
    .setMimeType(ContentService.MimeType.JSON);
}
```

## 手順4：ウェブアプリとして公開（デプロイ）する

「デプロイ」は、このプログラムに「届け先の住所（URL）」を発行する操作です。

1. 右上の **［デプロイ］→［新しいデプロイ］** をクリック
2. 左上の⚙（歯車）→ **［ウェブアプリ］** を選ぶ
3. 次のように設定します
   - **次のユーザーとして実行**：自分（自分のメールアドレス）
   - **アクセスできるユーザー**：**全員**（←児童のタブレットから送るために必要です）
4. **［デプロイ］** を押す
5. 「承認が必要です」と出たら **［アクセスを承認］** → 自分のアカウントを選ぶ
   - 「このアプリは確認されていません」と出た場合：**［詳細］→［（プロジェクト名）に移動］** を押せば進めます（自分で作ったプログラムなので大丈夫です）
6. 最後に表示される **ウェブアプリのURL**（`https://script.google.com/macros/s/…/exec`）を **コピー** します

## 手順5：アプリに送信先URLを設定する

1. アプリのHome画面 → **［🍎 先生用ログイン］** → 暗証番号 **2468**
2. 「📤 きろくの自動送信」の欄に、コピーしたURLを貼り付けて **［保存］**
3. **［接続テスト］** を押して「✅ 接続OK！」と出れば成功です

## 手順6：児童のタブレット全部に設定する

どちらかの方法で設定します。

- **方法A**：各タブレットで手順5と同じことをする（先生用ログイン→URL貼り付け→保存）
- **方法B（かんたん）**：アプリのURLのうしろに `?gas=コピーしたURL` を付けて一度開くと、自動で設定されます
  - 例：`index.html?gas=https://script.google.com/macros/s/…/exec`
  - このリンクをブックマークとして配れば、開くだけで設定完了です

---

## しくみと注意

- 児童が「保存する」を押すと、端末に保存 → すぐスプレッドシートへ送信されます
- **オフラインでも大丈夫**：送れなかったきろくは端末に貯まり、ネットにつながったときや次の保存時に自動でまとめて送られます（教師用画面の「未送信をいますぐ送る」でも送れます）
- 同じきろくが2回送られても、**IDで判定して1行しか書かれません**
- ⚠ **URLの取り扱い**：このURLを知っている人はスプレッドシートに書き込みができます。学級の外に広めないでください
- コードを貼り直したときは、**［デプロイ］→［デプロイを管理］→✏→バージョン「新バージョン」→デプロイ** で更新します（URLは変わりません）
